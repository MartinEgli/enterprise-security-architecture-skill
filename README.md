# enterprise-security-architecture-skill

Enterprise Security Architecture skill for secure design review, threat and risk
assessment, control mapping, security target architecture, Zero Trust direction,
and security governance.

## Install For Codex

```powershell
npx -y skills add MartinEgli/enterprise-security-architecture-skill --skill * -a codex --yes
```

## Checks

```powershell
npm run validate
npm run test
npm run package
```

## Continuous Improvement

This skill supports an explicit feedback loop. Use /enterprise-security-architecture feedback to
capture lessons and /enterprise-security-architecture improve to propose changes. Improvements must
remain traceable, validated, committed on a feature branch, and pushed before
they become part of the released skill behavior.
