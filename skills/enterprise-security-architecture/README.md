# Enterprise Security Architecture

Use this skill for secure design review, threat/risk/control mapping, and
security architecture decisions.

## Modes

- `/esa review`
- `/esa threat`
- `/esa controls`
- `/esa target`
- `/esa decision`
- `/esa clean-ai`
- `/esa diagram`

## Diagrams

Use `/esa diagram` for trust boundaries, security data flows, attack paths,
control maps, Zero Trust views, and AI security flows in Mermaid or PlantUML.
